# How to add custom assets

## Custom Songs

To make a custom song, simply drop an ogg file of the song, and a metadata file into externSongs:

`example-song.json`
```json
{
    "name": "Example Song",
    "artists": [
        "Example Artist"
    ],
    "album": "Example Album",
    "path": "path-from-songs-folder"
}
```
